놀라울 따름이다.

내가 무엇인가를 직접 만들지 않아도, 다른이에게 영감을 주거나 자극제가 되어 어떤 '의미있는' 일이 일어날 수 있다는 것.

물론, 그 사람이 꼭 내가 쓴 질문 글 때문에 만든 것은 아니겠지만, 간접적으로나마 영향을 준 것은 사실인 것 같다.

  

아이디어가 있을 때, 그것을 꼭꼭 숨겨두고 혼자만 생각해서는 결과물이 나오기가 어렵다.
